## 🚀 Project Overview

This is my personal portfolio, designed to showcase my projects, technical skills, and creativity. Built with React, TypeScript and Tailwind CSS, it features interactive UI, smooth animations, and a mobile-first responsive design.

## ✨ Features

- **Mobile-First Design:** Looks great on all devices.
- **Project Gallery:** Highlights selected projects using interactive cards.
- **Social & Contact Links:** Easy ways to connect, with animated and accessible UI
- **Modern, Minimal Design:** Clean, visually engaging layout with a focus on usability and aesthetics.
- **SEO & Analytics:** Optimised for search engines and includes privacy-friendly analytics.

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS, Vite
- **Animations:** Framer Motion, GSAP and custom CSS
- **Assets:** WEBP images, SVG icons, Google Fonts
- **Deployment:** Static site currently hosted via GitHub Pages

## 🧑‍💻 Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Run locally:**

   ```sh
   npm run dev
   ```

   Visit [http://localhost:5173](http://localhost:5173) in your browser.

3. **Build for production:**
   ```sh
   npm run build
   ```
